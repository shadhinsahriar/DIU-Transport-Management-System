# 🚍 DIU Transport Management System
Console-based Transport Management System developed in C using Doubly Linked Lists and File Handling. Features include route management, membership approval, complaint handling, payment tracking, transport notices, and role-based login systems for Admin, Student, Faculty, and Bus Staff.

This project was developed as an academic project for **Level 1 Term 3** at **Daffodil International University (DIU)**.

---

# 📌 Project Overview

The system is designed to manage university transport services digitally.  
It supports multiple user roles including:

- Admin
- Student
- Faculty
- Bus Staff (Driver/Conductor)

The project provides features for:

- Membership management
- Bus route management
- Complaint & feedback system
- Transport notices
- Emergency contacts
- Payment tracking
- User authentication

# 💾 File Handling

The project stores data using `.dat` files:

- `students.dat`
- `faculty.dat`
- `stuff.dat`
- `routes.dat`
- `reports.dat`
- `notice.dat`

These files preserve data even after closing the program.
